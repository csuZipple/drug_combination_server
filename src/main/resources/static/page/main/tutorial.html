Overview

DrugCombDB was created with the support of dcdb、nci and pmc database.

As you see on the homepage (the picture below), it shows the overview of our database Drug Combination Database (DrugCombDB). We provide a query input in every page to search the drug name in the right of the top menu. and if you are interested in some drugs but not sure the accurate name of the domain, you can also search the protein name of the domain ,Our system will give the relevant drug name as a reminder. The aside picture is a link that can show the application of our database.

Browser Page

After clicking the button, three browsing pictures will appear. The first one is to show the combination drug, the second is to show the target protein, and the last one shows a single drug. When you clicked on the target protein picture, it will scroll to the bottom of the page to display all drug information. At this time, you only need to select the corresponding drug to enter the target information page.

Combination Drug Data

The Integrate Data Format:

idfidsourcedrug1namedrug2nameconc1conc2growthcellline
11VemurafenibVemurafenib0.0000.0000.630635764944276A-375PMC5248573

Column descriptions:

  • id: key
  • fid: foreign key
  • source: the origin database
  • drug1name: drug A name
  • drug2name: drug B name
  • conc1: drug A concentration
  • conc2: drug B concentration
  • growth: Combined drug sensitivity score for this cancer cell line, between 0-1
  • cellline: The name of the cancer cell line

If you click on a drug name in the above table, you will get all the drug information made from this drug in the background. The display form consists of a scatter graph and a table. You can switch the display form by clicking the button in the upper right. You can also zoom in on the scatter graph to get more information by scrolling the mouse wheel. Click on each point to jump to this drug's detail page.
